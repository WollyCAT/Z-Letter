# Z-Letter Animator

Cute dark-mode web app that turns text into a downloadable handwriting-style animation.

## Features

- Separate language modes: English mode and Japanese mode (used independently).
- Expanded handwriting-style font collections for both English and Japanese.
- Character-by-character handwriting reveal preview.
- Adjustable writing speed, text color, font size, and stroke thickness.
- Preview controls with **Pause/Resume** while animation is running.
- Background options:
  - Pure white by default.
  - Pure color presets / custom pure color.
  - Imported image background.
- Download animation as `.webm`.

## Run

```bash
python -m http.server 4173
```

Then open <http://localhost:4173>.
