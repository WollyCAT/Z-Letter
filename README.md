# Z-Letter Animator

Cute dark-mode web app that turns text (English/Japanese) into a downloadable handwriting-style animation.

## Features

- Handwriting reveal animation for text, character-by-character.
- Supports English and Japanese text.
- Font selection with handwritten-friendly fonts.
- Adjustable writing speed, color, size, and stroke thickness.
- Background options:
  - Pure white by default.
  - Pure color presets / custom color.
  - Imported image background.
- Download animation as `.webm`.

## Run

```bash
python -m http.server 4173
```

Then open <http://localhost:4173>.
